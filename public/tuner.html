A tiny diffusion model (DDPM) was trained on layouts the project's adjacency rules produced, then asked to dream up new ones from pure noise. It learns only where each room wants to sit — on its own it outputs floating room centres (Raw). Feeding those into the rule-based placement snaps them into a clean, wall-sharing plan (Refined). The model proposes the arrangement; the rules render it.
