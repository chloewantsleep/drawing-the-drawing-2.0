dtd2 · diffusion samples

A tiny diffusion model (DDPM) was trained on layouts the project's adjacency rules produced, then asked to dream up new ones from pure noise. It learns only where each room wants to sit relative to the others — so on its own it outputs floating room centres (the Raw view). Feeding those into the project's rule-based placement snaps them into a clean, wall-sharing plan (the Refined view). The model proposes the arrangement; the rules render it.
